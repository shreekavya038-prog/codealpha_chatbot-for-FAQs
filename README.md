# CodeAlpha Chatbot for FAQs

A chatbot project developed using Python and Flask that answers frequently asked questions.

## Features
- Answers common questions automatically
- Simple and user-friendly interface
- Built with Python and Flask

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript

## How to Run
1. Install Python
2. Install Flask:
   pip install flask
3. Run the application:
   python app.py
4. Open http://127.0.0.1:5000/ in your browser
